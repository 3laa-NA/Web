# Organiz'Asso - Application Web

Application web pour la gestion des associations, permettant le partage de messages entre les membres et la communication privée.

## Prérequis

- Node.js v16+ et npm
- MongoDB v4.4+
- Un éditeur de code (VS Code recommandé)
